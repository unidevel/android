Welcome To Code Me Silly

To get started, save the sample file with an extension like HTML, or else it will load as a text file. All web developers should know that the extension means a lot to a server. It also means a lot to your browser and Code Me Silly's browser. Saving as .html instead of text could mean beautiful website instead of well... Text.

There are many operations that you do on Code Me Silly. Below is a guide for these operations.

Main Menu Operations

Pressing on menu on your device will show all the menu options.

Open

This menu option allows you to browse your file system and open specific files. You are allowed to open any file, so be careful not to open an image for editing, or else it will display a bunch on nonsense characters.

Save

This menu option allows you to save a file you have already saved and opened. This item allows you to save without browsing for the file already saved.

Save As

This menu option allows you to save a file as a new file. To save as a new file, Press Save As, then click menu and select Save As New File. This will allow you to specify a new name for the file in the current directory.

New

This menu option erases everything and allows you to save a new file

Preview

This menu option allows you to preview your currently working file. It is required that your file must be saved. If your file is not saved, you will be directed to your file system. Save as a new file, and you will be able to preview.

Copy and Paste

Code Me Silly had to jump through quite a few hoops to enable this feature. You are able to copy and paste only from this menu item. First you must copy to clipboard, then paste from this menu item.

Control Operations

Above your code is an option to show controls. After you touch this item, you will see more options

Search and Replace

Code Me Silly allows search and replace functionality. To perform a search, type the desired keyword and touch search. Once all the words are highlighted, you may replace all. To replace the highlighted words with a desired word, type the desired word into the second text box, and select replace all.

Code Highlighting Features

You may select a variety of highlighted code. Some of which are HTML, CSS, JavaScript, PHP, XML, Python, C, C++, Java, Clojure, CoffeeScript, Haskell, and Groovy.

Selecting A Theme

Code Me Silly understands that some users have been coding with dark backgrounds and light text. This can be quite and easy habit to follow, so Code Me Silly has provided different themes to suit everyones eyes.
